Introduction to HTML:

Definition: HTML (Hypertext Markup Language) is the standard markup language used to create and design web pages. It provides a structured format for presenting content on the internet. HTML uses various tags to define the structure and elements of a web page, such as headings, paragraphs, links, images, and more.

Explanation: HTML documents consist of elements enclosed in tags, which are interpreted by web browsers to display content. HTML tags are used to format and structure the content, and they are represented by angle brackets (< >). Tags can have attributes that provide additional information about the elements.

Example Program:1 
            <!DOCTYPE html>
            <html>
            <head>
                <title>My First Web Page</title>
            </head>
            <body>
                <h1>Hello, World!</h1>
                <p>This is my first web page created with HTML.</p>
            </body>
            </html>

Basic HTML: Definition: The basic HTML structure consists of the <!DOCTYPE html> declaration, the opening <html> tag, and the closing </html> tag. Inside the <html> tag, there are two sections: <head> and <body> . The <head> section contains meta-information about the document, while the <body> section contains the visible content of the web page.


Explanation: The <head> section is used to include metadata such as the document title, character encoding, CSS styles, and other elements that are not directly displayed on the page. The <body> section contains the visible content that users see when they visit the web page.


Document Body Text: Definition: In the <body> section, we can use various tags to define headings, paragraphs, and other text elements. HTML provides tags for different text formatting, such as bold, italic, underline, and more.


Explanation: By using appropriate text-related tags, we can structure and style the content of the web page for better readability and visual appeal.


Example Program:2 
        <!DOCTYPE html>
        <html>
        <body>
        <h1>Heading 1</h1>
        <h2>Heading 2</h2>
        <p>This is a paragraph.</p>
        <p>This is another paragraph.</p>
        <strong>This text is bold.</strong>
        <em>This text is italic.</em>
        </body>
        </html>
Hyperlinks:

Definition: Hyperlinks are created using the <a> tag with the href attribute, which specifies the URL of the link destination. Users can click on the link to navigate to the linked web page.

Explanation: Hyperlinks are an essential feature of the web, enabling users to navigate between different web pages and websites easily.

Example Program: 3 
<!DOCTYPE html>
    <html>
    <head>
        <title>My First Web Page</title>
    </head>

    <body>
        <h1>Welcome to My Website</h1>
        <p>Visit our <a href="https://www.profbharat.in"> My Website</a> for more information.</p>
    </body>
    </html>
Adding More Formatting:

Definition: HTML provides additional formatting tags like <mark> for highlighting text, <del> for deleted text (strikethrough), <ins> for inserted text (underlined), <sub> for subscript, and <sup> for superscript.

Explanation: These tags help in adding emphasis, marking changes, and displaying special text elements.

Example Program:4 
    <!DOCTYPE html>
    <html>
    <head>
        <title>My First Web Page</title>
    </head>

    <body>
        <h1>Welcome to My Website</h1>
        <p>This is a <mark>highlighted</mark> text.</p>
        <p>This is a <del>deleted</del> text and <ins>inserted</ins> text.</p>
        <p>This is a <sub>subscript</sub> and <sup>superscript</sup> text.</p>
    </body>
    </html>

Lists:

Definition: Lists can be represented using the <ul> (unordered list) or <ol> (ordered list) tags. Each list item is defined with the <li> tag.

Explanation: Lists help organize content into bullet points or numbered items for better readability and presentation.

Example Program:5 
<!DOCTYPE html>
    <html>
    <head>
        <title>My First Web Page</title>
    </head>

    <body>
        <h2>Unordered List</h2>
        <ul>
            <li>Item 1</li>
            <li>Item 2</li>
            <li>Item 3</li>
        </ul>
    
        <h2>Ordered List</h2>
        <ol>
            <li>First</li>
            <li>Second</li>
            <li>Third</li>
        </ol>
    </body>
    </html>
Tables:

Definition: Tables are created using the <table> tag. The table rows are defined with the <tr> tag, and table headers and data cells are represented by <th> and <td> tags, respectively.

Explanation: Tables are useful for organizing data into rows and columns, making it easier to compare and analyze information.

Images:

Definition: Images can be included in a web page using the <img> tag. The src attribute specifies the URL of the image, and the alt attribute provides alternative text that is displayed if the image cannot be loaded.

Explanation: Images enhance the visual appeal of a web page and provide additional context to the content.

Example Program:7 
    <!DOCTYPE html>
    <html>
    <head>
        <title>My First Web Page</title>
    </head>
    <body>
        <h2>Image Example</h2>
        <img src="image.jpg" alt="Sample Image">
    </body>

    </html>

Multimedia Objects:

Definition: Multimedia objects like videos and audio can be embedded using the <video> and <audio> tags, respectively. The controls attribute adds play, pause, and volume controls to the multimedia elements.

Explanation: Multimedia elements allow web pages to include videos and audio files, making the content more engaging and interactive.

Example Program:8 
    <!DOCTYPE html>
    <html>
    <head>
        <title>My First Web Page</title>
    </head>

    <body>
        <h2>Video Example</h2>
        <video controls>
            <source src="video.mp4" type="video/mp4">
            Your browser does not support the video tag.
        </video>
    
        <h2>Audio Example</h2>
        <audio controls>
            <source src="audio.mp3" type="audio/mpeg">
            Your browser does not support the audio tag.
        </audio>
    </body>

    </html>
Frames:

Definition: Frames allow a web page to be divided into multiple sections, each displaying a separate web page. The <frame> tag is used to define each individual frame and its content.

Explanation: Frames allow web developers to split a page into separate sections, each with its own content. However, frames are no longer recommended in modern web development due to their limitations and impact on usability and accessibility.

Example Program:9 
    <!DOCTYPE html>
    <html>
    <head>
        <title>My First Web Page</title>
    </head>

    <body>
        <h2>Frames Example</h2>
        <frame src="frame1.html" name="frame1">
        <frame src="frame2.html" name="frame2">
    </body>

    </html>
Forms:

Definition: Forms are used to collect user input. The <form> tag is used to create a form, and form elements like <input>, <select>, and <textarea> are used to capture user data. The action attribute specifies the URL to which the form data is sent, and the method attribute defines the HTTP method (GET or POST).

Explanation: Forms enable users to enter and submit data to web servers. They are widely used for various purposes, such as user registration, login, search, and more.

Example Program: 10 
    <!DOCTYPE html>
    <html>
    <head>
        <title>My First Web Page</title>
    </head>

    <body>
        <h2>Form Example</h2>
        <form action="submit.php" method="post">
            <label for="name">Name:</label>
            <input type="text" id="name" name="name" required><br>
            <label for="email">Email:</label>
            <input type="email" id="email" name="email" required><br>
            <input type="submit" value="Submit">
        </form>
    </body>

    </html>
